## 功能介绍

```properties
管理员
1、开发实现房屋租赁系统的整个系统程序； 
2、系统管理：个人中心、房屋信息管理、预约看房管理、合同信息管理、房屋报修管理、维修处理管理、房屋评价管理等模块的查看及相应操作；
3、房屋信息管理：实现房屋信息信息的查看及在线发布相应操作；
4、基础数据管理：实现房屋信息信息的添加、修改及删除等操作，并且根据需求进行房源信息的查看及发布预订等相应操作。
```
